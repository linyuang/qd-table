#### ※注意事项：<br>
##### * 该插件引用了webworker，若浏览器没有兼容该新特性将无法使用该插件。<br>
##### * 因浏览器不允许本地打开的文件访问webworker文件，所以开发者用代用运行调试功能的编译器进行插件测试(eg:idea,WebStorm)<br>
